# Geospatial-data-visualization-using-Sagemaker
This project builds a cloud-based geospatial visualization system using Amazon SageMaker. It enables interactive map creation, location marking, and region-based analysis using Python, GeoPandas, and Folium. Users can visualize geographic data and define areas of interest through coordinates for easier spatial analysis.
